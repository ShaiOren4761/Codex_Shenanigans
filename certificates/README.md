# Certificates

This folder is reserved for academic and professional credentials shown in the portfolio.

## Current Slots

- College Dean's List Certificate: placeholder page ready for the official file and exact details.
- Future technical certificates or course credentials.

## Add Before Publishing

For each certificate, add the real file and verify:

- Issuing organization or institution.
- Certificate title.
- Date, semester, or academic year.
- Whether the certificate should be view-only or downloadable.
