# Subscription Billing Mock App

A subscription billing admin mockup for plans, customers, invoices, payments, and failed-payment follow-up.

## Features

- Revenue, churn-risk, overdue, and active subscription metrics.
- Plan catalog with monthly and annual pricing.
- Customer table with search and status filters.
- Invoice list with payment states and retry workflow.
- Add new customers with plan assignment.
- Simulated billing event log.
- Local storage persistence with a reset option.

## Run

Open `index.html` in a browser.

## Portfolio Notes

This project demonstrates product-style admin software, billing-domain data modeling, tables, forms, status transitions, and practical dashboard UI.
