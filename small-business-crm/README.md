# Small Business CRM

A lightweight CRM for tracking contacts, companies, deals, tasks, and customer activity.

## Features

- Dashboard metrics for pipeline value, open deals, overdue tasks, and active contacts.
- Contact list with search and status filters.
- Deal pipeline grouped by stage.
- Add new contacts and deals.
- Task list with completion toggles.
- Customer activity timeline.
- Local storage persistence with a reset option.

## Run

Open `index.html` in a browser.

## Portfolio Notes

This project demonstrates CRUD-style workflows, business data modeling, dashboards, filtering, state updates, and a tidy user experience for repeated operational work.
