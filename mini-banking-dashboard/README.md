# Mini Banking Dashboard

A browser-based banking dashboard mockup with accounts, transactions, transfers, budgets, alerts, and local persistence.

## Features

- Multi-account overview with balances and utilization.
- Transfer money between accounts with validation.
- Search and filter transaction history.
- Budget progress by category.
- Simple risk alerts for unusual transactions and bill reminders.
- Local storage persistence with a reset option.

## Run

Open `index.html` in a browser.

## Portfolio Notes

This project demonstrates front-end application structure, realistic financial data modeling, user input validation, filtering, dashboard metrics, and polished responsive UI without relying on external packages.
